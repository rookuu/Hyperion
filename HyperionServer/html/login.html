Host: HyperionClient.py should be running on the Spotify computer. If it isn't, then turn it on now.

Guests: Push 'Add' to add songs to the playlist or 'Skip' to vote to skip the currently played song. Half the people connected (rounded up) must vote for a song to be skipped.
